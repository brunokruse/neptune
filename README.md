## jupyter tensorflow setup

### first, always upgrade pip!
```
pip install --upgrade pip
```

### install dependencies
```
pip install --upgrade ipython jupyter
pip install pandas
pip install numpy
pip install matplotlib
```

### make project folder
```
mkdir project-folder
cd project-folder
```

### run jupyter
```
jupyter notebook
```
